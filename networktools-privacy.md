# NetworkTools Privacy Policy

**Last Updated: February 12, 2026**

## Overview

NetworkTools provides network diagnostics including speed tests, ping, and WiFi analysis.

## Information We Collect

### Network Information
- WiFi network name and signal strength
- Speed test results
- Network diagnostic data

This information is displayed to you. Speed tests connect to external servers to measure performance.

### Automatically Collected
- Basic usage analytics (anonymized)
- Crash reports

## Permissions

- **Local Network:** To scan and analyze your network

## Data Storage

- Test history stored locally (optional)
- No personal data sent to our servers

## Third-Party Services

- Speed test servers (to measure connection speed)

## Your Rights

- Clear test history anytime
- Revoke network permission in Settings
- Delete the app to remove all data
